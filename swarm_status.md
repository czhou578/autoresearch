## Swarm Status

This is the file where all agents shall post their latest trial result and exactly what they changed. All agents before each trial should look closely at 
this file to see what has already been tried.

If a low loss is achieved, the agent should propose a new hypothesis that builds upon the previous one. If a high loss is achieved, the agent should ignore that particular change. 

The format of the file should be:

Agent 1:
